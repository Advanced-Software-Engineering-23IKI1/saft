# FFT und Rendering Prototyp

## Begründung
Es wurde die kritische Komponente der Umwandlung einer `.wav` Audiodatei in ein `.png` Bild umgesetzt. Dies wurde ausgewählt da es einerseits komplex ist und essentiell ist für die funktionsweise von SAFT.

## Installation
getestet und ausgeführt auf
- npm version   >= 11.6.2
- node version  >= v24.13.0

````bash
npm install
````


## Ausführung
````bash
node prototype.js -f ./input/test.wav 
````

output ist in `./output`
